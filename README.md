 ## 功能
 - 编辑：恢复、适应屏幕、左图适应、右图适应
 - 效果：亮度、相框、色温
 - 变换：缩放、翻转、旋转、逆时针旋转、顺时针旋转
 - 模糊：简单模糊、高斯模糊
 - 增强：中值滤波、拉普拉斯锐化
 - 特殊：灰度、轮廓提取、二值化、腐蚀、膨胀、开运算、闭运算
